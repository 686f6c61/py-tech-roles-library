        Guía de uso · 01

        Empezar

        Una clase, carga perezosa: no toca disco hasta la primera consulta. Las salidas de los comentarios son reales, ejecutadas contra la versión publicada.

        python
        from tech_roles_library import TechRoles

lib = TechRoles(language="en")

role = lib.get_role("BE-L3")
role.role                  # Backend Developer
role.level                 # L3 - Junior II
role.core_competencies[0]  # Mastery of RESTful API design and best practices
        from tech_roles_library import TechRoles

        Guía de uso · 02

        Dos idiomas, mismos códigos

        Cada idioma es un catálogo completo: nombres, competencias y niveles localizados. Los códigos funcionan como identificador estable entre ambos.

        python
        en = TechRoles(language="en")
es = TechRoles(language="es")

en.get_role("PT-L5").role   # Penetration Tester
es.get_role("PT-L5").role   # Pentester

es.roles()[:3]
# ('Administrador de Bases de Datos', 'Administrador de Sistemas', 'Agile Coach')

        Guía de uso · 03

        Consultar roles

        Por código o por nombre y nivel. El nivel acepta 3, "3", "L3" o el nombre completo "L3 - Junior II".

role.years_range    # YearsRange(min=5, max=7)

lib.get_role_by_name("Backend Developer", 3)
[e.code for e in lib.get_levels("Data Engineer")]
# ['DE-L1', 'DE-L2', ..., 'DE-L9']
        role = lib.get_role("AAG-L5")             # por código, sin distinguir mayúsculas

lib.get_role_by_name("Desarrollador de Backend", 3)

        Guía de uso · 04

        Competencias

        Cada rol-nivel tiene competencias core, complementarias e indicadores observables. Y una vista acumulada desde L1, sin duplicados.

        python
        c = lib.get_competencies("Backend Developer", "L3")
len(c.core), len(c.complementary), len(c.indicators)   # (8, 3, 3)
c.indicators[0]   # Completes complex features with minimal supervision

acc = lib.get_accumulated("Backend Developer", 5)
len(acc.core)     # 40 — todo lo dominado de L1 a L5
        c = lib.get_competencies("Desarrollador de Backend", "L3")
len(c.core), len(c.complementary), len(c.indicators)   # (8, 3, 3)
c.indicators[0]   # Completa funcionalidades complejas con mínima supervisión

acc = lib.get_accumulated("Desarrollador de Backend", 5)
len(acc.core)     # 40 — todo lo dominado de L1 a L5

        Guía de uso · 05

        Progresión de carrera

        Vista completa de carrera, salto al siguiente nivel con las competencias nuevas, y nivel recomendado por años de experiencia.

        python
        path = lib.get_career_path("Backend Developer", "L5")
[e.level_number for e in path.mastered]   # [1, 2, 3, 4]
[e.level_number for e in path.growth]     # [6, 7, 8, 9]

nxt = lib.get_next_level("Backend Developer", "L3")
nxt.next.code               # BE-L4
nxt.new_competencies[0]     # Ability to design distributed systems and microservices

lib.get_by_experience("Backend Developer", 5).level   # L4 - Mid-Level I
        path = lib.get_career_path("Desarrollador de Backend", "L5")

        Guía de uso · 08

        Errores que ayudan

        Toda excepción hereda de TechRolesError, y los "no encontrado" sugieren alternativas.

        python
        from tech_roles_library import RoleNotFoundError

try:
    lib.get_role_by_name("Bakcend Developer", 3)
except RoleNotFoundError as e:
    print(e)
# Role not found: 'Bakcend Developer'.
# Did you mean: Backend Developer, Game Developer, Blockchain Developer?

lib.validate_role("Backend Developer")   # True
lib.validate_level("Backend Developer", 10)   # False
